# STAT 184 Homework 4.3 and 4.4 Repository

## What this repository is for:
This repository contains my work for STAT 184 Homework 4.3
The main focus of the project is to create a reproducible data analysis report using Quarto,
and have all the needed materials for reproducibility avalable on a github repository than anyone can close one to start the reproduciblity.

## Data:
This project uses:
- scrabed data from a publicly available busiest airport passenger traffic data from Wikipedia
    Used for the airport analysis in HW 3.4
- simulated/generated data for the Monte Carlo numeric integration section
- `calcium.csv` for the calcium treatment data
    Shared with ChatGPT to test its ability in wrangling data

## Current Plan:
My current plan is to keep all files for the assignment organized in one public repository, including the source QMD file, the rendered PDF, the dataset, and any other files needed to support reproducibility.
I also want to use this repo to demonstrate the GitHub skills we learned about in class, including:
    - Creating and maintaining a public repository
    - Using branches
    - Making meaningful commits
    - Creating and closing issues
    - Opening pull requests
    - Keeping the main branch updated

## Repository Organization:
- `hw-4-3-reproducible-analysis.qmd` – Quarto source file
- `hw-4-3-reproducible-analysis.pdf` – rendered PDF output
